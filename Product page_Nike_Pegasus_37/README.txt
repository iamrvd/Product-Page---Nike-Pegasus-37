Nike Pegasus 37 - Product Page

This is a responsive product page created for Nike Pegasus 37 shoes. All design assets have been created from scratch. Adobe tools like Illustrator, XD and Cinema4D have been used for creating the assets.
3D model of the shoe has also been implemented. Using GreenSock library, Hotspots have been placed on the 3D model. Promotional video for the product has also been added. Application scales from mobile to Desktop

Technologies Used:
Vanilla JS, CSS, Adobe Tools

Instructions:

1. Go to project folder using the change directory (cd) command in the terminal.
2. Run the command:
	docker-compose run
3. To view the front-end, type: localhost:3792 in the url.
